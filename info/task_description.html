Help Stephen to automate his task with creating module. As you could see on illustration, gray circle means on, and white circle means off. every digit in time string, contains different slot. First digit for hour contains 2 length. Second digit for hour contains 4 length. First digit for minute and second, contains 3 length. And second digit for minute and hour, contains 4 length. Every digit in time is converted to binary representation. Then, for ease of this task, you will converted every on ( or 1 ) signal to dash ( - ) and every off ( or 0 ) signal to dot ( . )

Input: A time string ( could be hh:mm:ss or h:m:s format )

Output: A string of Morse symbol from binary representation with specific format( "h h : m m : s s" format )

Example: 

assert checkio( "10:37:49" ) == "._ .... : .__ .___ : _.. _.._", "First Test"
assert checkio( "21:34:56" ) == "-. ...- : .-- .-.. : -.- .--.", "Second Test"
assert checkio( "00:01:02" ) == ".. .... : ... ..._ : ... ..-.", "Third Test"
assert checkio( "23:59:59" ) == "-. ..-- : -.- -..- : -.- -..-", "Fourth Test"
